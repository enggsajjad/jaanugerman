09 telc Prüfungsformat

A2

A2 telc Deutsch A2 — Prüfungsformat / Exam Format

Hinweis / Note: Grammatik, Wortschatz und Fertigkeiten sind für Goethe und telc identisch — nur das Prüfungsformat unterscheidet sich. Die Dateien 01–07 in diesem Ordner gelten für beide Prüfungen.

Grammar, vocabulary, and skills are identical for Goethe and telc — only the exam format differs. Files 01–07 in this folder apply to both exams.

Überblick / Overview

telc Deutsch A2 Goethe A2 (zum Vergleich)
Dauer / Duration ca. 70 Minuten (schriftlich) + 11 Min (mündlich) ca. 80 Minuten + 15 Min
Teile / Parts Hören, Lesen, Schreiben, Sprechen Lesen, Hören, Schreiben, Sprechen
Bestehen / Pass 60% gesamt 60% pro Modul
Preis / Price ~100–150 € ~100–150 €

Wichtiger Unterschied / Key Difference: Wie bei A1 — bei telc zählt die Gesamtpunktzahl. Eine Schwäche kann durch andere Module ausgeglichen werden.

As with A1 — telc counts the total score. A weakness can be compensated by other modules.

HÖREN / LISTENING (ca. 20 Minuten)

Teil 1: Alltagsgespräche / Everyday Conversations (5 Aufgaben / items)

Sie hören fünf kurze Gespräche aus dem Alltag. Zu jedem Gespräch gibt es eine Frage mit drei Antwortmöglichkeiten.

You hear five short everyday conversations. Each has one question with three answer options.

Aufgabentyp / Task type: Multiple Choice (a, b, c) mit Bildern / with pictures

Beispiel / Example:

Sie hören ein Gespräch an der Bushaltestelle. Frage: Wann kommt der nächste Bus? a) In 5 Minuten ☐ b) In 10 Minuten ☐ c) In 20 Minuten ☐ → b) ✓

Tipps / Tips: - Bilder genau ansehen VOR dem Hören. Look at the pictures carefully BEFORE listening. - Beim ersten Hören die richtige Antwort markieren, beim zweiten überprüfen. Mark answer on first listen, verify on second.

Teil 2: Durchsagen und Ansagen / Announcements (5 Aufgaben / items)

Sie hören Durchsagen am Bahnhof, im Kaufhaus, im Radio usw. Entscheiden Sie: Richtig oder Falsch?

You hear announcements at the station, department store, radio, etc. Decide: True or False?

Aufgabentyp / Task type: Richtig / Falsch

Beispiel / Example:

Sie hören: "Wegen Bauarbeiten fährt die S-Bahn Linie 3 nur bis Hauptbahnhof. Bitte steigen Sie dort in den Bus um." Aussage: Die S-Bahn fährt normal. → Falsch

Teil 3: Telefonische Nachrichten / Phone Messages (5 Aufgaben / items)

Sie hören Nachrichten auf dem Anrufbeantworter und ergänzen Informationen in einem Notizzettel.

You hear voicemail messages and complete information in a note.

Aufgabentyp / Task type: Lückentext / Gap fill

Beispiel / Example:

Anruf von: Praxis Dr. Schneider Neuer Termin: Donnerstag, _ Uhr → 15:00 Bitte mitbringen: ___ → Versichertenkarte

Tipps / Tips: - Notizzettel vorher lesen — Sie wissen dann, worauf Sie achten müssen. Read the note beforehand — you'll know what to listen for. - Oft gefragt: Uhrzeiten, Telefonnummern, Adressen, Namen. Often asked: times, phone numbers, addresses, names.

LESEN / READING (ca. 20 Minuten)

Teil 1: Kurztexte verstehen / Understanding Short Texts (5 Aufgaben / items)

Sie lesen kurze Texte aus Zeitungen, Anzeigen oder dem Internet und beantworten Fragen.

Read short texts from newspapers, ads, or the internet and answer questions.

Aufgabentyp / Task type: Multiple Choice (a, b, c)

Beispiel / Example:

Anzeige: „Spanischkurs für Anfänger. Jeden Dienstag 18–19:30 Uhr. Volkshochschule Köln. Anmeldung bis 15. September." Frage: Wann beginnt der Kurs? a) Am Montag ☐ b) Am Dienstag ☐ c) Am Mittwoch ☐ → b) ✓

Teil 2: Texte und Informationen zuordnen / Matching Texts and Information (5 Aufgaben / items)

Sie lesen Situationsbeschreibungen und ordnen sie passenden Texten (z.B. Anzeigen, Veranstaltungen) zu.

Read situation descriptions and match them to appropriate texts (e.g., ads, events).

Aufgabentyp / Task type: Zuordnung / Matching

Beispiel / Example:

Situation: Ihre Tochter (8 Jahre) möchte schwimmen lernen. Welche Anzeige passt? → Anzeige C: Schwimmkurs für Kinder 6–10 Jahre, Hallenbad Süd

Teil 3: Einen längeren Text verstehen / Understanding a Longer Text (5 Aufgaben / items)

Sie lesen einen zusammenhängenden Text (Brief, E-Mail, Zeitungsartikel) und beantworten Fragen.

Read a continuous text (letter, email, newspaper article) and answer questions.

Aufgabentyp / Task type: Richtig / Falsch

Tipps / Tips: - Fragen zuerst lesen, dann den Text. Read questions first, then the text. - Antworten stehen in der Reihenfolge des Textes. Answers appear in text order. - Vorsicht bei "nicht", "nur", "aber", "trotzdem". Watch out for "not", "only", "but", "nevertheless".

SCHREIBEN / WRITING (ca. 20 Minuten)

Teil 1: Kurze Nachricht / Short Message (1 Aufgabe, ca. 30–40 Wörter)

Schreiben Sie eine kurze Nachricht, E-Mail oder SMS zu einer Alltagssituation. Sie bekommen 3 Inhaltspunkte.

Write a short message, email, or SMS about an everyday situation. You get 3 content points.

Beispiel / Example:

Situation: Ihr Nachbar hat ein Paket für Sie angenommen. Sie waren nicht zu Hause. Schreiben Sie an Ihren Nachbarn: - Bedanken Sie sich. / Thank them. - Wann holen Sie das Paket ab? / When will you pick up the package? - Bitten Sie um etwas. / Ask for something.

Musterantwort / Model answer:

Lieber Herr Schmidt, vielen Dank, dass Sie mein Paket angenommen haben! Ich möchte es morgen Abend abholen, gegen 18 Uhr. Ist das OK für Sie? Können Sie es bitte vor Ihre Tür stellen, falls Sie nicht da sind? Viele Grüße, [Name]

Teil 2: Formular ausfüllen / Fill in a Form (1 Aufgabe)

Füllen Sie ein Formular mit persönlichen Daten aus (z.B. Anmeldung für einen Kurs, Bibliotheksausweis).

Fill in a form with personal details (e.g., course registration, library card).

Typische Felder / Typical fields: - Vorname, Nachname, Geburtsdatum, Geburtsort - Adresse (Straße, PLZ, Ort) - Telefonnummer, E-Mail - Staatsangehörigkeit, Beruf - Unterschrift, Datum

Tipps / Tips: - PLZ = Postleitzahl (z.B. 50667 für Köln). Zip code. - Geburtsdatum: TT.MM.JJJJ — Date of birth: DD.MM.YYYY - Unterschreiben Sie mit Ihrem richtigen Namen! Sign with your real name!

SPRECHEN / SPEAKING (ca. 11 Minuten, Gruppenprüfung)

Wichtig: Wie bei A1 — die telc Sprechprüfung ist eine Gruppenprüfung mit einem Partner.

Important: As with A1 — the telc speaking exam is a group exam with a partner.

Teil 1: Über sich sprechen / Talk About Yourself (ca. 2 Min)

Sie stellen sich vor und beantworten Fragen der Prüfer. Detaillierter als bei A1!

Introduce yourself and answer examiner questions. More detailed than A1!

Erwartete Themen / Expected topics: - Name, Alter, Herkunft — Name, age, origin - Familie und Kinder — Family and children - Beruf / Ausbildung / Studium — Job / training / studies - Wohnort und Wohnung — City and apartment - Tagesablauf — Daily routine - Freizeit und Hobbys — Free time and hobbies - Sprachen lernen — Learning languages

Nützliche Sätze / Useful phrases: - Ich lerne seit ... Monaten Deutsch. — I've been learning German for ... months. - Morgens stehe ich um ... auf und dann... — In the morning I get up at ... and then... - Am liebsten ... ich in meiner Freizeit. — I like to ... most in my free time. - Ich wohne in einer Wohnung mit ... Zimmern. — I live in an apartment with ... rooms.

Teil 2: Über ein Thema sprechen / Talk About a Topic (ca. 3 Min)

Sie bekommen eine Karte mit einem Thema und Bildern. Beschreiben Sie, was Sie sehen und erzählen Sie von eigenen Erfahrungen.

You get a card with a topic and pictures. Describe what you see and share your own experiences.

Typische Themen / Typical topics: - Einkaufen und Essen — Shopping and food - Gesundheit und Arztbesuch — Health and doctor visits - Wohnen und Nachbarschaft — Living and neighborhood - Verkehr und Reisen — Transport and travel - Schule und Lernen — School and learning

Nützliche Sätze / Useful phrases: - Auf der Karte / Auf dem Bild sehe ich... — On the card / In the picture I see... - Ich denke, das Thema ist... — I think the topic is... - In meinem Land / Bei uns ist das anders, weil... — In my country it's different because... - Ich finde es wichtig, dass... — I think it's important that...

Teil 3: Gemeinsam etwas planen / Plan Something Together (ca. 6 Min)

Sie und Ihr Partner bekommen eine Aufgabe und planen gemeinsam. Sie müssen Vorschläge machen, reagieren und sich einigen.

You and your partner get a task and plan together. Make suggestions, react, and agree.

Beispiel / Example:

Ein Freund aus dem Deutschkurs hat Geburtstag. Planen Sie zusammen eine Überraschungsparty. A friend from German class has a birthday. Plan a surprise party together. - Wo? / Where? - Wann? / When? - Essen und Trinken? / Food and drinks? - Geschenk? / Gift? - Wen noch einladen? / Who else to invite?

Nützliche Sätze / Useful phrases: - Was meinst du? / Was meinen Sie? — What do you think? - Wie wäre es mit...? — How about...? - Das ist eine gute Idee, aber... — That's a good idea, but... - Ich bin einverstanden. — I agree. - Können wir vielleicht...? — Can we maybe...? - Dann machen wir das so! — Then let's do it that way! - Wer bringt was mit? — Who brings what?

Bewertung / Scoring

Bereich / Section Punkte / Points
Hören 45
Lesen 45
Schreiben 30
Sprechen 30
Gesamt / Total 150
Bestehen / Pass ≥ 90 (60%)

Schriftlich und mündlich müssen separat bestanden werden: - Schriftlich (Hören + Lesen + Schreiben): ≥ 72 von 120 (60%) - Mündlich (Sprechen): ≥ 18 von 30 (60%)

Written and oral must be passed separately: - Written (Listening + Reading + Writing): ≥ 72 of 120 (60%) - Oral (Speaking): ≥ 18 of 30 (60%)

Unterschiede Goethe vs. telc A2 / Differences

Goethe A2 telc A2
Sprechen Einzelprüfung Gruppenprüfung (2 Teilnehmer)
Bestehen 60% pro Modul 60% schriftlich + 60% mündlich
Schreiben 2 Teile: Nachricht + Antwort 2 Teile: Nachricht + Formular
Hören 4 Teile 3 Teile
Lesen 4 Teile 3 Teile

Tipps für die telc A2-Prüfung / Tips for the telc A2 Exam

  1. Partner-Sprechen üben — Finden Sie einen Übungspartner. Reagieren Sie auf Vorschläge! Practice partner speaking. React to suggestions!
  2. Formulare schnell ausfüllen — Üben Sie mit echten Formularen (Bibliothek, VHS, Arzt). Practice with real forms (library, adult education, doctor).
  3. Wortschatz thematisch — telc fragt besonders oft: Einkaufen, Gesundheit, Wohnen, Arbeit. telc especially asks about: shopping, health, living, work.
  4. Hörstrategien — Notizzettel vorher lesen, Schlüsselwörter markieren. Read notes beforehand, mark key words.
  5. Zeitmanagement — Schreiben Sie nicht zu lange! 30–40 Wörter reichen für Teil 1. Don't write too long! 30–40 words are enough for Part 1.
  6. Nummern und Adressen — Üben Sie das Hören und Schreiben von Zahlen, Uhrzeiten, Adressen. Practice hearing and writing numbers, times, addresses.
