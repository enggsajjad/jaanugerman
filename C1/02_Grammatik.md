# C1 Grammatik / Grammar

Fachkundige Grammatik für das Goethe-Zertifikat C1.

---

## 1. Erweiterte Partizipialkonstruktionen

### Links-Attribute (typisch für wissenschaftliche Texte)
- Die **im Rahmen der Studie durchgeführten** Experimente ... (= Experimente, die im Rahmen der Studie durchgeführt wurden)
- Die **von zahlreichen Experten als überholt angesehene** Theorie ... (= Theorie, die von zahlreichen Experten als überholt angesehen wird)
- Der **sich seit Jahren verschärfende** Fachkräftemangel ...

### Umformung: Relativsatz ↔ Partizipialattribut
| Relativsatz | Partizipialattribut |
|-------------|-------------------|
| Die Maßnahmen, die von der Regierung beschlossen wurden | Die von der Regierung beschlossenen Maßnahmen |
| Die Kosten, die ständig steigen | Die ständig steigenden Kosten |
| Die Studie, die kürzlich veröffentlicht wurde | Die kürzlich veröffentlichte Studie |
| Die Fragen, die noch zu klären sind | Die noch zu klärenden Fragen |

## 2. Nomen-Verb-Verbindungen (Funktionsverbgefüge)

| Wendung | Bedeutung | Beispiel |
|---------|-----------|---------|
| Anklage erheben | anklagen | Gegen den Verdächtigen wurde Anklage erhoben. |
| Aufsehen erregen | auffallen | Der Fall erregte großes Aufsehen. |
| Auskunft erteilen | informieren | Können Sie mir Auskunft erteilen? |
| Berücksichtigung finden | berücksichtigt werden | Alle Aspekte müssen Berücksichtigung finden. |
| Bezug nehmen auf | sich beziehen auf | Ich nehme Bezug auf Ihr Schreiben. |
| Einfluss nehmen auf | beeinflussen | Die Medien nehmen Einfluss auf die öffentliche Meinung. |
| in Angriff nehmen | beginnen | Das Projekt wurde in Angriff genommen. |
| in Betracht ziehen | erwägen | Wir sollten alle Optionen in Betracht ziehen. |
| in Frage stellen | bezweifeln | Diese Theorie wird zunehmend in Frage gestellt. |
| Kritik üben an | kritisieren | Die Opposition übt scharfe Kritik an der Regierung. |
| Stellung nehmen zu | sich äußern zu | Der Minister nahm Stellung zu den Vorwürfen. |
| unter Beweis stellen | beweisen | Sie hat ihre Kompetenz unter Beweis gestellt. |
| zur Debatte stehen | diskutiert werden | Dieses Thema steht zur Debatte. |
| zur Verfügung stehen | vorhanden sein | Stehen Ihnen genügend Mittel zur Verfügung? |
| zum Ausdruck bringen | ausdrücken | Er brachte seine Besorgnis zum Ausdruck. |

## 3. Modalpartikeln / Modal Particles

| Partikel | Funktion | Beispiel |
|----------|----------|---------|
| ja | Betonung (bekannt) | Das ist **ja** interessant! |
| doch | Widerspruch, Aufforderung | Komm **doch** mit! / Das stimmt **doch** nicht! |
| halt/eben | Resignation | Das ist **halt/eben** so. |
| wohl | Vermutung | Er ist **wohl** schon gegangen. |
| schon | Zuversicht/Einschränkung | Das wird **schon** klappen. / Das stimmt **schon**, aber ... |
| mal | Abschwächung | Können Sie **mal** helfen? |
| bloß/nur | Warnung/Nachdrücklichkeit | Tu das **bloß** nicht! |
| denn | Interesse/Überraschung | Was ist **denn** passiert? |
| eigentlich | Themenübergang | Was machst du **eigentlich** beruflich? |
| ruhig | Ermutigung | Du kannst **ruhig** fragen. |
| etwa | Ungläubige Frage | Hast du **etwa** vergessen? |

## 4. Subjektlose Passivkonstruktionen

- Es **wurde** bis spät in die Nacht **diskutiert**.
- Hier **darf** nicht **geraucht werden**.
- Im Vertrag **wird** festgelegt, dass ...
- **Wird** hier noch **gearbeitet**?
- Es **wurde** ihm mitgeteilt, dass ...

## 5. Konjunktiv I — Indirekte Rede (Vertiefung)

### Bildung

| Infinitiv | er/sie (K1) | sie (Pl.) (K1→K2) |
|-----------|-------------|-------------------|
| sein | sei | seien |
| haben | habe | hätten (K2, da K1=haben=Indikativ) |
| können | könne | könnten |
| müssen | müsse | müssten |
| gehen | gehe | gingen |
| kommen | komme | kämen |

### Anwendung in der Berichterstattung
- Der Sprecher erklärte, die Regierung **sei** sich der Probleme bewusst.
- Die Forscherin betonte, es **gebe** deutliche Hinweise auf einen Zusammenhang.
- Er fügte hinzu, man **müsse** weitere Untersuchungen **durchführen**.
- Laut Bericht **seien** die Ergebnisse **veröffentlicht worden**.
- Die Ministerin sagte, sie **werde** die Maßnahmen **überprüfen lassen**.

## 6. Satzbau — Komplexe Strukturen

### Verbletzt-Sätze mit multiplen Nebensätzen
- Obwohl die Studie zeigt, **dass** die Maßnahmen wirksam **sind**, **bleibt** unklar, **ob** sie langfristig finanzierbar **sein werden**.

### Korrelate (es)
- **Es** ist davon auszugehen, **dass** sich die Lage verschärft.
- **Es** liegt nahe, **dass** der Rückgang auf wirtschaftliche Faktoren zurückzuführen ist.
- **Es** bedarf weiterer Forschung, **um** diesen Zusammenhang zu bestätigen.

### Satzverschachtelung
- Die These, **die** der Autor, **der** an der Universität Heidelberg lehrt, aufgestellt hat, **ist** umstritten.
→ Besser: Die vom Heidelberger Professor aufgestellte These ist umstritten.

## 7. Irreale Vergleiche / Unreal Comparisons

- Er tut so, **als ob** er alles wüsste. / Er tut so, **als** wüsste er alles.
- Sie sprach, **als ob** sie eine Expertin **wäre**.
- Es sieht aus, **als ob** es regnen **würde**.
- Er tat, **als hätte** er nichts gehört.

---

## 8. Weiterführende Relativsätze

### Mit "was" (bezieht sich auf den ganzen Satz)
- Er hat die Prüfung bestanden, **was** mich sehr freut.
- Sie kam zu spät, **was** alle überrascht hat.
- Er hat gelogen, **was** ihm niemand verziehen hat.

### Mit "worüber, wofür, worin" usw.
- Das ist ein Thema, **worüber** viel diskutiert wird.
- Das ist ein Ergebnis, **worauf** wir stolz sein können.

---

## 9. Verbalstil vs. Nominalstil

### Verbalstil (mündlich, informell)
- Man hat die Kosten **gesenkt**.
- Die Bevölkerung **wächst**.
- Man **muss berücksichtigen**, dass ...

### Nominalstil (schriftlich, formell)
- **Die Senkung** der Kosten wurde durchgeführt.
- **Das Wachstum** der Bevölkerung.
- **Unter Berücksichtigung** der Tatsache, dass ...
