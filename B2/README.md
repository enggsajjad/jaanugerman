# B2 — Goethe-Zertifikat B2

## Übersicht / Overview

Das Goethe-Zertifikat B2 bestätigt ein fortgeschrittenes Sprachniveau. Sie können die Hauptinhalte komplexer Texte verstehen und sich spontan und fließend verständigen.

**Empfohlene Lernzeit**: 600–800 Unterrichtsstunden (kumulativ)

## Prüfungsstruktur

| Modul | Dauer | Punkte | Bestehen |
|-------|-------|--------|----------|
| Lesen | 65 min | 100 | min. 60 |
| Hören | 40 min | 100 | min. 60 |
| Schreiben | 75 min | 100 | min. 60 |
| Sprechen | 15 min | 100 | min. 60 |
| **Gesamt** | **195 min** | **400** | **min. 240** |
