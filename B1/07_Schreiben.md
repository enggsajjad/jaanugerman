# B1 Schreiben / Writing

Schreibübungen zur Vorbereitung auf das Goethe-Zertifikat B1.
*Writing exercises to prepare for the Goethe-Zertifikat B1.*

---

## Prüfungsformat / Exam Format

| Teil / Part | Aufgabe / Task | Punkte / Points |
|------|---------|--------|
| 1 | Persönliche E-Mail (ca. 80 Wörter) / Personal email (~80 words) | 40 |
| 2 | Meinung äußern: Diskussionsbeitrag (ca. 80 Wörter) / Express opinion: discussion post (~80 words) | 40 |
| 3 | Formelle Nachricht (ca. 40 Wörter) / Formal message (~40 words) | 20 |

**Dauer / Duration: 60 Minuten / minutes — 100 Punkte / points**

---

## Teil 1: Persönliche E-Mail / Personal Email

### Übung 1 / Exercise 1
**Aufgabe / Task:** Ihr Freund Stefan hat seinen Job verloren. Schreiben Sie ihm. *Your friend Stefan lost his job. Write to him.*
- Mitgefühl ausdrücken / Express sympathy
- Was würden Sie raten? / What would you advise?
- Eigene Erfahrung / Own experience

**Musterantwort / Sample answer:**
> Lieber Stefan,
> das tut mir wirklich leid! Ich kann mir vorstellen, dass das eine schwierige Situation ist. Aber jede Krise ist auch eine Chance.
> An deiner Stelle würde ich die Zeit nutzen, um über meine Zukunft nachzudenken. Vielleicht wäre eine Weiterbildung gut? Als ich vor zwei Jahren meinen Job verloren habe, habe ich einen Computerkurs gemacht, und das hat mir sehr geholfen.
> Lass den Kopf nicht hängen!
> Liebe Grüße, [Name]

*Dear Stefan, I'm really sorry! I can imagine this is a difficult situation. But every crisis is also an opportunity. In your place, I would use the time to think about the future. Maybe further training would be good? When I lost my job two years ago, I took a computer course, and that really helped me. Keep your chin up! Best, [Name]*

## Teil 2: Diskussionsbeitrag / Discussion Post

### Struktur für Meinungstexte / Structure for Opinion Texts

1. **These / Thesis** — Eigene Position klar nennen / State your position clearly
2. **Argumente / Arguments** — 2–3 Gründe / 2–3 reasons
3. **Gegenargument / Counter-argument** — Andere Seite kurz erwähnen / Briefly mention other side
4. **Fazit / Conclusion** — Zusammenfassung / Summary

### Redemittel / Useful Phrases

| Funktion / Function | Deutsch | Englisch |
|---------|---------|----------|
| These / Thesis | Ich bin der Meinung, dass ... | I am of the opinion that ... |
| Argument | Erstens ... / Ein wichtiges Argument ist ... | Firstly ... / An important argument is ... |
| Gegenargument / Counter | Andererseits ... / Man muss aber bedenken ... | On the other hand ... / But one must consider ... |
| Beispiel / Example | Zum Beispiel ... / Ein gutes Beispiel ist ... | For example ... / A good example is ... |
| Fazit / Conclusion | Zusammenfassend ... / Alles in allem ... | In summary ... / All in all ... |

## Teil 3: Formelle Nachricht / Formal Message

### Übung / Exercise
**Aufgabe / Task:** Sie haben einen kaputten Drucker bestellt. Schreiben Sie an den Kundenservice. *You ordered a broken printer. Write to customer service.*

**Musterantwort / Sample answer:**
> Sehr geehrte Damen und Herren,
> am 5. März habe ich einen Drucker bestellt (Bestellnr. 789456). Leider funktioniert er nicht — er druckt nur leere Seiten. Ich möchte das Gerät umtauschen oder eine Erstattung erhalten.
> Mit freundlichen Grüßen, [Name]

*Dear Sir or Madam, on March 5th I ordered a printer (order no. 789456). Unfortunately it doesn't work — it only prints blank pages. I would like to exchange the device or receive a refund. Sincerely, [Name]*

## Tipps / Tips

1. **Alle Inhaltspunkte bearbeiten.** *Address ALL content points.* — Für jeden Punkt gibt es Punkte! *Each point earns marks!*
2. **Wortanzahl beachten.** *Watch the word count.* — Nicht zu kurz, nicht zu lang. *Not too short, not too long.*
3. **Textsorte beachten.** *Mind the text type.* — Teil 1: du, Teil 3: Sie. *Part 1: informal, Part 3: formal.*
4. **Abwechslungsreiche Grammatik.** *Varied grammar.* — Nebensätze, Konjunktiv II, Passiv zeigen. *Show subordinate clauses, Subjunctive II, passive.*
5. **Korrekturlesen!** *Proofread!* — 5 Minuten am Ende reservieren. *Reserve 5 minutes at the end.*
