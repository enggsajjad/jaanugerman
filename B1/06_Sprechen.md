# B1 Sprechen / Speaking

Übungen zur Vorbereitung auf den mündlichen Teil des Goethe-Zertifikats B1.
*Exercises to prepare for the speaking part of the Goethe-Zertifikat B1.*

---

## Prüfungsformat / Exam Format

| Teil / Part | Aufgabe / Task | Dauer / Duration |
|------|---------|-------|
| 1 | Gemeinsam etwas planen / Plan something together | ca. 3 Min |
| 2 | Ein Thema präsentieren / Present a topic | ca. 3-4 Min |
| 3 | Feedback geben und Fragen stellen / Give feedback and ask questions | ca. 2 Min |

**Paarprüfung / Pair exam: 2 Teilnehmer / participants, ca. 15 Minuten / minutes**

---

## Teil 1: Gemeinsam planen / Plan Together

### Redemittel / Useful Phrases

| Deutsch | Englisch |
|---------|----------|
| Ich schlage vor, dass wir ... | I suggest that we ... |
| Wie wäre es, wenn wir ...? | How about if we ...? |
| Wir könnten doch ... | We could ... |
| Das finde ich eine gute Idee! | I think that's a good idea! |
| Ja, einverstanden! | Yes, agreed! |
| Das finde ich nicht so gut, weil ... | I don't think that's so good because ... |
| Ich würde lieber ... | I would rather ... |

## Teil 2: Kurzvortrag / Short Presentation

### Struktur (3–4 Minuten) / Structure (3–4 minutes)

1. **Einleitung** / *Introduction* — Thema nennen, warum wichtig / Name topic, why important
2. **Eigene Erfahrung** / *Personal experience* — Persönliches Beispiel / Personal example
3. **Situation im Heimatland** / *Situation in your home country*
4. **Vor- und Nachteile** / *Advantages and disadvantages*
5. **Eigene Meinung und Schluss** / *Own opinion and conclusion*

### Redemittel / Useful Phrases

**Einleitung / Introduction:**
- Ich möchte heute über das Thema "..." sprechen. *(I would like to talk about the topic "..." today.)*
- Dieses Thema ist sehr aktuell, weil ... *(This topic is very current because ...)*

**Eigene Erfahrung / Personal experience:**
- Ich selbst ... / In meinem Fall ... *(I myself ... / In my case ...)*

**Situation im Heimatland / Home country:**
- In meinem Heimatland ... *(In my home country ...)*

**Vor- und Nachteile / Pros and cons:**
- Einerseits ... andererseits ... *(On the one hand ... on the other hand ...)*
- Ein Vorteil ist, dass ... *(An advantage is that ...)*
- Ein Nachteil ist, dass ... *(A disadvantage is that ...)*

**Eigene Meinung / Own opinion:**
- Meiner Meinung nach ... *(In my opinion ...)*
- Ich finde, dass ... *(I think that ...)*
- Zusammenfassend möchte ich sagen ... *(In summary, I would like to say ...)*

### Beispielthemen / Example Topics
- Soll man Kinder streng erziehen? / Should children be raised strictly?
- Stadt oder Land — wo lebt man besser? / City or countryside — where is life better?
- Braucht man heutzutage noch ein Auto? / Does one still need a car nowadays?
- Ist Homeoffice besser als Büroarbeit? / Is home office better than office work?

## Teil 3: Feedback und Fragen / Feedback and Questions

### Redemittel / Useful Phrases

| Deutsch | Englisch |
|---------|----------|
| Vielen Dank für deinen Vortrag. | Thank you for your presentation. |
| Ich fand besonders interessant, dass ... | I found it particularly interesting that ... |
| Bei uns ist es ähnlich / anders, weil ... | With us it's similar / different because ... |
| Ich habe eine Frage: ... | I have a question: ... |
| Ich stimme dir zu, dass ... | I agree with you that ... |
