09 telc Prüfungsformat

A1

A1 telc Deutsch A1 — Prüfungsformat / Exam Format

Hinweis / Note: Grammatik, Wortschatz und Fertigkeiten sind für Goethe und telc identisch — nur das Prüfungsformat unterscheidet sich. Die Dateien 01–07 in diesem Ordner gelten für beide Prüfungen.

Grammar, vocabulary, and skills are identical for Goethe and telc — only the exam format differs. Files 01–07 in this folder apply to both exams.

Überblick / Overview

telc Deutsch A1 Goethe A1 (zum Vergleich)
Dauer / Duration ca. 75 Minuten ca. 80 Minuten
Teile / Parts Hören, Lesen, Schreiben, Sprechen Lesen, Hören, Schreiben, Sprechen
Bestehen / Pass 60% gesamt 60% pro Modul
Preis / Price ~100–130 € ~100–150 €
Ergebnis / Result ca. 4–6 Wochen ca. 4–6 Wochen

Wichtiger Unterschied / Key Difference: Bei telc müssen Sie 60% insgesamt erreichen, nicht pro Modul. Das bedeutet, eine Schwäche in einem Bereich kann durch Stärke in einem anderen ausgeglichen werden.

With telc you need 60% overall, not per module. A weakness in one area can be compensated by strength in another.

HÖREN / LISTENING (ca. 20 Minuten)

Teil 1: Kurze Ansagen / Short Announcements (6 Aufgaben / items)

Sie hören kurze Ansagen aus dem Alltag (Bahnhof, Supermarkt, Telefon). Zu jedem Hörtext gibt es eine Aufgabe.

You hear short everyday announcements (train station, supermarket, phone). Each has one task.

Aufgabentyp / Task type: Richtig oder Falsch / True or False

Beispiel / Example:

Sie hören: "Der Zug nach München fährt heute von Gleis 3, nicht von Gleis 7." Aussage: Der Zug fährt von Gleis 7. → Falsch

Tipps / Tips: - Lesen Sie die Aussage VOR dem Hören. Read the statement BEFORE listening. - Achten Sie auf Verneinungen (nicht, kein). Watch for negations. - Sie hören jeden Text zweimal. You hear each text twice.

Teil 2: Kurze Gespräche / Short Dialogues (4 Aufgaben / items)

Sie hören kurze Gespräche zwischen zwei Personen. Wählen Sie die richtige Antwort (a, b oder c).

You hear short dialogues between two people. Choose the correct answer (a, b, or c).

Aufgabentyp / Task type: Multiple Choice (3 Optionen)

Beispiel / Example:

Sie hören ein Gespräch im Café. Frage: Was bestellt die Frau? a) Kaffee ☐ b) Tee ☐ c) Orangensaft ☐ → a) Kaffee ✓

Teil 3: Telefonansagen / Phone Messages (5 Aufgaben / items)

Sie hören Nachrichten auf dem Anrufbeantworter. Ergänzen Sie die fehlenden Informationen.

You hear voicemail messages. Complete the missing information.

Aufgabentyp / Task type: Lückentext / Gap fill (Zahlen, Uhrzeiten, Namen)

Beispiel / Example:

Termin beim Arzt: Mittwoch um _____ Uhr. Sie hören: "Ihr Termin ist am Mittwoch um halb drei." → 14:30

Tipps / Tips: - Üben Sie Zahlen und Uhrzeiten! Practice numbers and times! - Schreiben Sie sofort mit. Write immediately while listening.

LESEN / READING (ca. 25 Minuten)

Teil 1: Schilder und Kurztexte / Signs and Short Texts (5 Aufgaben / items)

Sie lesen kurze Texte wie Schilder, Anzeigen, Hinweise. Entscheiden Sie: Richtig oder Falsch?

Read short texts like signs, ads, notices. Decide: True or False?

Aufgabentyp / Task type: Richtig / Falsch

Beispiel / Example:

Schild: „Öffnungszeiten: Mo–Fr 9–18 Uhr, Sa 9–14 Uhr" Aussage: Am Samstag kann man bis 18 Uhr einkaufen. → Falsch

Teil 2: Kurze Texte zuordnen / Matching Short Texts (5 Aufgaben / items)

Sie lesen kurze Texte (z.B. Kleinanzeigen) und ordnen sie den passenden Situationen zu.

Read short texts (e.g., classified ads) and match them to situations.

Aufgabentyp / Task type: Zuordnung / Matching

Beispiel / Example:

Situation: Sie suchen eine Wohnung mit 2 Zimmern. a) „Großes WG-Zimmer frei, ab sofort" — Nein b) „2-Zimmer-Wohnung, 55qm, ab 01.05." — Ja ✓

Teil 3: Formular oder Anzeige / Forms or Ads (5 Aufgaben / items)

Sie lesen ein Formular, eine Anzeige oder einen kurzen Brief und beantworten Fragen.

Read a form, ad, or short letter and answer questions.

Aufgabentyp / Task type: Multiple Choice (3 Optionen)

Tipps / Tips: - Schlüsselwörter in der Frage markieren. Mark key words in the question. - Antwort steht fast immer wörtlich im Text. The answer is almost always literally in the text.

SCHREIBEN / WRITING (ca. 20 Minuten)

Teil 1: Formular ausfüllen / Fill in a Form (1 Aufgabe)

Sie füllen ein einfaches Formular aus (Name, Adresse, Telefonnummer, Beruf, etc.).

Fill in a simple form (name, address, phone number, job, etc.).

Beispiel / Example:

Feld / Field Ihre Angaben / Your details
Vorname _____
Nachname _____
Geburtsdatum _____
Adresse _____
Telefon _____
E-Mail _____
Beruf _____

Tipps / Tips: - Achten Sie auf: Vorname ≠ Nachname. Watch out: first name ≠ last name. - Datum im deutschen Format: TT.MM.JJJJ (z.B. 15.03.1990). Date in German format.

Teil 2: Kurze Nachricht schreiben / Write a Short Message (1 Aufgabe)

Schreiben Sie eine kurze Nachricht oder SMS (ca. 30 Wörter) zu einer Alltagssituation.

Write a short message or SMS (approx. 30 words) about an everyday situation.

Beispiel / Example:

Situation: Sie können morgen nicht zum Deutschkurs kommen. Schreiben Sie eine Nachricht an Ihre Lehrerin. You can't come to German class tomorrow. Write a message to your teacher.

Punkte / Points: - Alle 3 Inhaltspunkte bearbeiten. Address all 3 content points. - Anrede und Gruß nicht vergessen. Don't forget greeting and sign-off. - 30 Wörter reichen! 30 words are enough!

Musterantwort / Model answer:

Liebe Frau Müller, ich kann morgen leider nicht zum Kurs kommen. Ich bin krank. Kann ich die Hausaufgaben per E-Mail bekommen? Vielen Dank! Mit freundlichen Grüßen, [Name]

SPRECHEN / SPEAKING (ca. 11 Minuten, Gruppenprüfung)

Wichtig: Die telc Sprechprüfung ist eine Gruppenprüfung (2–3 Teilnehmer), NICHT eine Einzelprüfung wie bei Goethe.

Important: The telc speaking exam is a group exam (2–3 participants), NOT an individual exam like Goethe.

Teil 1: Sich vorstellen / Introduce Yourself (ca. 2 Min)

Stellen Sie sich vor. Die Prüfer zeigen Ihnen Karten mit Themen.

Introduce yourself. The examiners show you cards with topics.

Themen / Topics: - Name, Alter, Herkunft / Name, age, origin - Wohnort / Where you live - Sprachen / Languages - Beruf oder Ausbildung / Job or education - Familie / Family - Hobbys / Hobbies

Nützliche Sätze / Useful phrases: - Ich heiße... / Mein Name ist... — My name is... - Ich komme aus... — I come from... - Ich wohne in... — I live in... - Ich spreche... — I speak... - Ich bin... von Beruf. — I am a... (job). - Ich bin verheiratet / ledig. — I am married / single. - Mein Hobby ist... / In meiner Freizeit... — My hobby is... / In my free time...

Teil 2: Bilder beschreiben und Gespräch / Describe Pictures and Conversation (ca. 4 Min)

Sie bekommen ein Bild und sprechen darüber. Dann stellen Sie Ihrem Partner / Ihrer Partnerin eine Frage.

You get a picture and talk about it. Then ask your partner a question.

Themen / Topics: - Einkaufen, Familie, Essen, Wohnung, Verkehr, Freizeit - Shopping, family, food, home, transport, free time

Nützliche Sätze / Useful phrases: - Auf dem Bild sehe ich... — In the picture I see... - Das ist ein/eine... — That is a... - Die Person macht/hat... — The person is doing/has... - Und du? / Und Sie? — And you? - Wie findest du...? — What do you think of...?

Teil 3: Etwas aushandeln / Negotiate Something (ca. 5 Min)

Sie und Ihr Partner planen etwas zusammen. Machen Sie Vorschläge und reagieren Sie auf die Vorschläge des Partners.

You and your partner plan something together. Make suggestions and react to your partner's suggestions.

Beispiel / Example:

Sie möchten zusammen kochen. Besprechen Sie: - Was kochen? / What to cook? - Wann? / When? - Was einkaufen? / What to buy? - Wer macht was? / Who does what?

Nützliche Sätze / Useful phrases: - Wollen wir...? / Sollen wir...? — Shall we...? - Ich schlage vor, dass... — I suggest that... - Gute Idee! / Das finde ich gut! — Good idea! - Ich bin nicht sicher. Vielleicht... — I'm not sure. Maybe... - Einverstanden! / OK, machen wir das. — Agreed! / OK, let's do that.

Bewertung / Scoring

Bereich / Section Punkte / Points
Hören 45
Lesen 45
Schreiben 30
Sprechen 30
Gesamt / Total 150
Bestehen / Pass ≥ 90 (60%)

Wichtig / Important: - Schriftliche Prüfung (Hören + Lesen + Schreiben) und mündliche Prüfung (Sprechen) können an verschiedenen Tagen stattfinden. - Written exam (Listening + Reading + Writing) and oral exam (Speaking) can take place on different days. - Es gibt kein Bestehen pro Modul — nur die Gesamtpunktzahl zählt! - There is no passing per module — only the total score counts!

Unterschiede Goethe vs. telc A1 / Differences

Goethe A1 telc A1
Sprechen Einzelprüfung + Paar Gruppenprüfung (2–3)
Bestehen 60% pro Modul 60% gesamt
Schreiben Formular + kurze Nachricht Formular + kurze Nachricht
Hören 3 Teile 3 Teile
Lesen 3 Teile 3 Teile
Allgemein Etwas variabler Vorhersehbarer / More predictable

Tipps für die telc A1-Prüfung / Tips for the telc A1 Exam

  1. Formular korrekt ausfüllen — Üben Sie das deutsche Datumsformat und Ihre persönlichen Daten. Practice the German date format and your personal details.
  2. Zahlen und Uhrzeiten — Diese kommen in fast jedem Hörtext vor! These appear in almost every listening text!
  3. Gruppenprüfung üben — Üben Sie mit einem Partner, Fragen zu stellen und zu reagieren. Practice with a partner asking questions and responding.
  4. 30 Wörter reichen — Schreiben Sie nicht zu viel. Kurz und korrekt ist besser als lang und fehlerhaft. Don't write too much. Short and correct beats long and faulty.
  5. Schlüsselwörter markieren — Markieren Sie wichtige Wörter in Fragen und Texten. Mark key words in questions and texts.
